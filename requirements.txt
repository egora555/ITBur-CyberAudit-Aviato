# Системные требования
# 
# Программа требует следующие системные утилиты:
# - iproute2 (предоставляет команду 'ss' для сетевого аудита)
# - dpkg (для аудита пакетов в Debian-системах)
# - systemd (для проверки статуса сервисов)
# - python3-tk (для графического интерфейса)
#
# В Kali Linux всё это уже предустановлено.
# Если чего-то не хватает, установите командой:
# sudo apt-get update
# sudo apt-get install iproute2 dpkg systemd python3-tk
#
#
# Зависимости Python
#
# Программа использует ТОЛЬКО стандартную библиотеку Python!
# Никаких дополнительных pip пакетов не требуется.
#
# Протестировано на:
# - Kali Linux (на виртуальной машине, предоставленой для кейса)
# - Python 3.12
#
#
# Для других Debian-систем:
# sudo apt-get install python3 python3-tk iproute2 dpkg
#
# Для систем с RPM (Fedora, CentOS, RHEL):
# Примечание: аудит пакетов (dpkg) НЕ будет работать на RPM-системах
