## Описание проекта

**Aviato Auditor** — инструмент для автоматического аудита безопасности Linux-систем. Программа выполняет комплексную проверку системы и выдаёт понятные рекомендации по устранению найденных уязвимостей.

## Возможности

- **Анализ прав доступа** — поиск файлов с опасными правами (777, 666) и секретами
- **Сетевой аудит** — сканирование открытых портов через `ss -tulpn` и анализ опасных сервисов
- **Аудит пакетов** — проверка версий установленного ПО на известные уязвимости (CVE)
- **Рекомендации** — для каждой проблемы выдаётся конкретная команда по устранению
- **Сохранение результатов** — экспорт отчётов в текстовый файл

## Требования к системе

- **Операционная система:** Linux (Kali Linux / Ubuntu / Debian)
- **Python:** версия 3.12 или выше
- **Системные утилиты** (обычно уже установлены в Kali Linux):
  - `iproute2` — для команды `ss`
  - `dpkg` — для аудита пакетов
  - `systemd` — для проверки статуса сервисов
  - `python3-tk` — для графического интерфейса

## Установка и запуск

### 1. Клонирование репозитория

    git clone https://github.com/egora555/ITBur-CyberAudit-Aviato.git
    cd ITBur-CyberAudit-Aviato

### 2. Установка системных зависимостей

**Kali Linux / Debian / Ubuntu:**

    sudo apt-get update
    sudo apt-get install python3 python3-tk iproute2 dpkg

> В Kali Linux большинство зависимостей уже предустановлено.

### 3. Запуск программы

**Базовый запуск** (без аудита при старте):

    sudo python3 auditor.py

> `sudo` необходима для полного аудита — без прав root программа не может выполнять сетевой аудит и полный анализ прав, так как часть файлов и директорий недоступна.

**Справка по командам:**

    python3 auditor.py --help

**Сканирование ключевых директорий** (`/etc`, `/var`, `/home`):

    python3 auditor.py --scan key

**Сканирование конкретной директории:**

    python3 auditor.py --scan "/путь/к/директории"

> Рекомендуется всегда оборачивать путь в кавычки — без них директории с пробелами в имени будут распознаны некорректно.

**Сканирование всей системы:**

    sudo python3 auditor.py --scan all

**Сетевой аудит:**

    python3 auditor.py --network

> Для сетевого аудита требуются права root.

**Аудит установленных пакетов:**

    python3 auditor.py --package

**Сохранение результатов в файл:**

    python3 auditor.py --output "/путь/к/файлу.txt"

> Команда `--output` используется совместно с другими флагами. Путь может быть как абсолютным, так и относительным. Файл должен существовать до запуска — если файл отсутствует, стартовый аудит не выполнится.
> Программа автоматически создаёт output-файл по директории 'output/[индекс] ДД_ММ_ГГГГ/Ч_М_С' если не указывать output

**Полный аудит с сохранением результатов:**

    sudo python3 auditor.py --scan key --network --package --output results/result.txt

## Возможные проблемы

**`ModuleNotFoundError: No module named 'tkinter'`**
    sudo apt-get install python3-tk

    > Если программа всё ещё не запускается, попробуйте создать venv через python3 -m venv

**`ERROR: Не удалось выполнить команду ss`**

    sudo apt-get install iproute2

**Нет информации о процессах в сетевом аудите**

    sudo python3 auditor.py --network
